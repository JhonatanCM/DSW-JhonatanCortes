Framework Bootstrap

¿Qué es bootstrap y para qué sirve?

¿Qué haremos?

Ejemplos

Con tan solo llamar a las siguientes librerías de bootstrap desde nuestro head, ya podremos utilizar cualquiera de las ventajas que este nos ofrece. Veamos cómo, a continuación:

<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
<!-- jQuery library -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<!-- Popper JS -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
<!-- Latest compiled JavaScript -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

BOTONES

Podríamos agregar cualquiera de los botones a continuación, solo modificando el class atributo por el correspondiente al nombre del botón.

<button type="button" class="btn btn-primary">Primary</button>

RESULTADO:

ALERTAS

También llamando al class atributo correspondiente podremos agregar alertas.

<div class="alert alert-success">
  <strong>Success!</strong> Indicates a successful or positive action.
</div>

RESULTADO:

MODAL

Entre otras características también nos ofrece la posibilidad de insertar modal o pop-up.

BARRA DE NAVEGACIÓN

Por último, podemos resaltar que nos permite agregar barras de navegación y muchos otros complementos más, que no podría abarcar en este post.
